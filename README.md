Ficsúr Gergely - Portfólió

Ez a személyes portfólió oldalam, amely bemutatja a fejlesztői tudásomat, projektjeimet és szakmai filozófiámat. A célom egy modern, reszponzív és biztonságos felület létrehozása volt, amely tükrözi a technológiai fejlődés iránti elkötelezettségemet.
🚀 Technológiai Stack

    Frontend: HTML5, Modern CSS (Tailwind koncepciók), JavaScript (ES6+)

    Dizájn: Minimalista, "Dark Mode" központú, glassmorphism elemekkel

    Biztonság: Content Security Policy (CSP), Input-szanálás, biztonságos külső hivatkozások kezelése

    Verziókezelés: Git & GitHub

🛡️ Biztonsági Szemlélet (Security by Design)

Junior fejlesztőként kiemelt figyelmet fordítok az alkalmazásaim biztonságára:

    CSP (Content Security Policy): Szigorú irányelvek a cross-site scripting (XSS) támadások megelőzésére.

    Link védelem: Minden külső hivatkozás rel="noopener noreferrer" attribútummal van ellátva a biztonságos böngésző-szigetelés érdekében.

    Spam-védelem: Az e-mail címek dinamikus, JavaScript alapú megjelenítése, amely hatékonyan védi a kapcsolatfelvételi adatokat az automatizált botoktól.
